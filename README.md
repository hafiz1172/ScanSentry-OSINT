# 🛡️ ScanSentry OSINT Tool

**ScanSentry** is a professional, Python-based Open Source Intelligence (OSINT) tool designed for security researchers and investigators. It provides a clean GUI to perform rapid reconnaissance on any domain or IP address.

## ✨ Features
* **DNS Lookup:** Quickly find the IP address of any domain.
* **Geo-IP Tracking:** Locate the physical server location, City, Country, and ISP.
* **WHOIS Research:** Fetch domain registration details, creation, and expiration dates.
* **Threaded Execution:** The GUI remains responsive even during heavy data fetching.
* **Cyber-Dark UI:** A sleek, hacker-style interface for a professional feel.
